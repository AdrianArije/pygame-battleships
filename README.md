# pygame-battleships
An interactive Battleships game developed as part of a team project.

## Features
- Player vs AI (Easy, Medium, Hard)
- Ship placement system with drag & rotate
- Turn-based gameplay
- Hit/miss tracking and win detection
- Sound effects and UI

## Controls
- Mouse: Place ships / select tiles
- R while holding left click down: Rotate ship
- Space: Fire shot
- TAB: Return to main menu

## Tech Used
- Python
- Pygame


## How to Run
1. Install pygame:
   pip install pygame
2. Run:
   python main.py
